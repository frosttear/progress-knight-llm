# Progress Knight LLM Edition

### Introduction
This is a fork of [Progress Knight](https://github.com/ihtasham42/progress-knight) by Ihtasham42, enhanced with an AI-powered story system and various gameplay and UI improvements.

### What is Progress Knight about?
Progress Knight is a life-sim incremental game set in a fantasy/medieval world. You start as a beggar and work your way up through jobs and skills across multiple reincarnated lives, carrying forward max-level bonuses each time you rebirth.

---

### Changes from the Original Game

#### Story System (New)
- **Milestone stories** — at key job/skill levels (1, 10, 25, 50, 100, 200, 500, 1000), a narrative story is generated describing your character's journey. Stories are generated by an LLM (bring your own API key) or fall back to a library of pre-written stories.
- **Life review** — when you rebirth, a summary story of the life you just lived is shown. The review pauses the game and is attributed to the correct ending life, including the actual age at death.
- **Story effects** — each milestone and life review grants a bonus effect (XP multiplier, income bonus, happiness boost, or lifespan bonus). Effects from rebirths are always **permanent**; milestone effects may be temporary or permanent depending on your life number.
- **Active effects panel** — a new sidebar section shows all currently active story bonuses, grouped by category (XP / Income / Happiness / Lifespan) with collapsible sections showing totals and individual effect details including source job and level.
- **Story log** — a dedicated tab shows the full history of stories across all lives, filterable by life number.

#### Rebirth Improvements
- **`rebirthTwo` max-level bug fixed** — the original game incorrectly reset `maxLevel` to 0 for all tasks after a type-2 rebirth, making the max-level XP bonus permanently stuck at ×1.0. This is now fixed; max levels carry forward correctly through both rebirth types.

#### Localization (New)
- Full **Chinese (Simplified)** and **English** UI support with auto-detection based on browser language.
- All UI strings, story effects, notifications, and guide text are localized.

#### UI Improvements
- **Responsive mobile layout** — the game adapts to smaller screens.
- **Quick task display** — shows current job and skill at a glance in the sidebar.
- **Detailed multiplier breakdown** — clicking the XP or income gain on any task row shows a full breakdown of every active multiplier (max-level bonus, happiness, skills, story effects, equipment), making it easy to see exactly where your gains come from.
- **In-game guide tab** — explains mechanics, formulas, and career path strategy in both languages.
- **Dark/light mode** preserved and extended to new UI elements.
- **Save export/import** uses Unicode-safe Base64 encoding for cross-browser compatibility.

#### Settings
- LLM story generation can be enabled/disabled; supports configurable API endpoint, model, and your own API key (BYOK).
- Language can be switched at any time from the settings panel.
